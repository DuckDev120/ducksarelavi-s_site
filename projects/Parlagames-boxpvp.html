ParlaGames Update - BoxPVP

About the Project

As a development partner on the major re-release of the BoxPvP game mode, I was responsible for implementing core gameplay changes, balancing systems, and refining the overall player experience. Using Java, the Paper API, and structured data storage via MySQL, I developed the updated health mechanic that grants temporary hearts on kills, redesigned the progression and leveling system for more accurate and fair scaling, and integrated new kits across all ranks with dynamic configuration handling. I also worked on introducing rotating maps—including one adapted from the lobby—alongside a new loot-chest system featuring visual key items, reward logic, and probability-based drop mechanics, including items that increase key-drop chances. Throughout the update, I handled essential bug fixes such as ensuring kill detection works properly with End Crystals and TNT, preventing effects from persisting after death, and implementing a reliable system that preserves selected kits even after relogging. The work required precise balancing, extensive QA, performance optimization, and close collaboration with the team to ensure all new features integrated seamlessly into the existing architecture while delivering a smoother, more polished BoxPvP experience.
